# Dominion — A Visual Journey

A portrait-mode reading experience through *Dominion: How the Christian Revolution Remade the World* by Tom Holland.

Each chapter is distilled into 12 key points, accompanied by classical artworks and historical maps.

## Chapter I — Athens: The Making of the Western Mind

A scroll-through visual essay tracing the ancient world's attempts to ground power in cosmic order — from Persian empire to Greek reason to Roman law — and how each vision ultimately served the strong, setting the stage for Christianity.

## Stack

- React + Vite
- Cormorant Garamond & Cinzel typefaces
- Scroll-driven animations via Intersection Observer

## Development

```bash
npm install
npm run dev
```

## Design

Dark antiquity palette with gold accents. Optimised for portrait reading at a comfortable text measure. Each slide reveals with staggered fade-in animations as you scroll.
